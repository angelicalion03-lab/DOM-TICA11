⚙️ Componentes de la Domótica

Un sistema domótico es como un organismo vivo: tiene órganos sensoriales que perciben el entorno.

Está formado por 4 tipos de componentes:

🔵 SENSOR
Detecta condiciones del entorno y envía señales al controlador. Ej: PIR, temperatura, humo.
→
🟡 CONTROLADOR
Recibe señales, procesa con reglas programadas y envía órdenes. Ej: Arduino, Raspberry Pi, Hub.
→
🟢 ACTUADOR
Ejecuta la acción física: enciende luces, abre puertas, regula temperatura. Ej: bombilla, cerradura.
🔴 INTERFAZ DE USUARIO — controla todo el sistema ↑
Permite al humano ver y controlar el sistema. Ej: app móvil, Alexa, panel táctil, dashboard web.
