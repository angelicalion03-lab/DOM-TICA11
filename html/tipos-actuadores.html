Tipos actuadores

Tipos de actuadores

Bombilla inteligente
Recibe ordenes de encendido, apagado y cambio de color o intensidad via WiFi o Zigbee.
Ejemplo: Philips Hue, Govee RGBIC.
Circuito: Controlador → señal WiFi → bombilla → luz encendida.
Cerradura inteligente
Abre y cierra la puerta por comando digital: app, código PIN o reconocimiento de huella.
Ejemplo: Yale Assure Lock, Kwikset Halo.
Integración: Se puede automatizar con el sensor de movimiento exterior.
Termostato inteligente
Regula la calefacción y el aire acondicionado según la temperatura deseada.
Ejemplo: Google Nest, Ecobee SmartThermostat.
Ahorro: Reduce el consumo energético hasta un 23%.
Electroválvula de riego
Abre y cierra el paso de agua para regar jardines o cultivos automáticamente.
Ejemplo: Electroválvula 12V DC con módulo relay Arduino.
Sensor asociado: Sensor de humedad del suelo — riega solo cuando es necesario.
